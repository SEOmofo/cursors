Cursors by SEOmofo

Custom cursors and CSS for SEOs and web developers.

These cursor icons and CSS can be used with browser extensions like Stylish to customize your web experience. The examples I've provided are particularly useful for SEOs who need a simple way to check whether or not a given link is likely to be counted in Google's PageRank algorithm.

Nofollow

Add nofollow.css to your custom stylesheet to identify links that have the rel="nofollow" attribute.

JavaScript

Add javascript.css to your custom stylesheet to identify links that have an href attribute value that starts with javascript:, or links that have an onclick attribute but the href attribute is either empty (href="") or a hash character (href="#").
